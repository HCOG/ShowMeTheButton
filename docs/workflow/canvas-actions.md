# 画布操作按钮 (btn-add-node, btn-copy, btn-paste, btn-delete)

**位置**: 画布上方工具栏
**类型**: 画布编辑按钮
**功能**: 编辑画布上的节点

## 按钮说明

### ➕ 添加节点 (btn-add-node)
快速添加一个新节点（默认是任务节点）。

### 📋 复制 (btn-copy)
复制当前选中的节点。复制后可以粘贴到画布其他位置。

### 📄 粘贴 (btn-paste)
粘贴已复制的节点到画布。

### 🗑️ 删除 (btn-delete)
删除当前选中的节点及其连接。

## 节点操作（鼠标悬停时显示）

### ✏️ 编辑节点 (btn-edit-{id})
打开节点的属性编辑面板，可以修改名称、描述等。

### 📋 复制节点 (btn-copy-{id})
复制当前节点到剪贴板。

### 🗑️ 删除节点 (btn-delete-{id})
删除当前节点。

## 使用方法

#### 复制粘贴节点
1. 选中要复制的节点（点击节点）
2. 点击"📋 复制"按钮或按 Ctrl+C
3. 点击"📄 粘贴"按钮或按 Ctrl+V
4. 复制的节点会出现在原节点附近

#### 删除节点
1. 选中要删除的节点
2. 点击"🗑️ 删除"按钮
3. 系统会确认删除操作
