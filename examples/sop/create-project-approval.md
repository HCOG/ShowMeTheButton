# Create Project and Submit for Approval

## Purpose
Create a new project record and send it to the manager for approval.

## Scope
This procedure is for requesters and managers in the project management workspace.

## Prerequisites
- User is signed in.
- User has requester permission.
- Required budget code is available.

## Procedure
1. Go to `/projects` and open the Projects page.
2. Click "New Project" to start a project request.
3. Fill the required "Project Name" and budget code fields.
4. If the project budget is over 10000, attach the budget justification document.
5. Review all project details for accuracy.
6. Click "Submit for Approval". After successful submission, the status should become "Pending Approval".

## Troubleshooting
If a required field is missing, complete the highlighted field and retry submission.
