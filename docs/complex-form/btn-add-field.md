# 表单导航按钮 (btn-prev, btn-next, btn-submit)

**位置**: 表单底部
**类型**: 导航操作按钮组
**功能**: 在表单步骤之间导航或提交

## 按钮说明

### ◀ 上一步 (btn-prev)
返回到上一步。第一步时禁用。

### 下一步 ▶ (btn-next)
进入下一步。最后一步不显示此按钮。

### ✓ 提交表单 (btn-submit)
提交整个表单。最后一步时显示。

## 使用方法

### 多步骤提交流程
1. 在第一步填写基本信息
2. 点击"下一步 ▶"进入第二步
3. 在第二步填写地址信息
4. 点击"下一步 ▶"进入第三步
5. 在第三步填写就业信息
6. 点击"下一步 ▶"进入第四步
7. 在第四步上传证件并选择偏好
8. 阅读并同意服务条款
9. 点击"✓ 提交表单"完成提交

## 状态说明

- 提交按钮在未同意服务条款时禁用
- 每一步都有进度条显示
- 必填字段未填写时无法进入下一步

## 常见用户查询

- "怎么进入下一步？"
- "我要提交表单"
- "上一步在哪？"
- "我还没填完"
- "怎么确认提交？"
