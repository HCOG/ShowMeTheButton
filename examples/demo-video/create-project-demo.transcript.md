# Create Project and Submit for Approval

This demo shows a requester creating a project and sending it to a manager for approval.

Prerequisites:
- The requester is signed in.
- The requester has project creation permission.

The user opens the Projects page, starts a new project, fills the project name, and submits it.
If the budget is over 10000, the requester should attach justification before submission.
After successful submission, the project status should become Pending Approval.
