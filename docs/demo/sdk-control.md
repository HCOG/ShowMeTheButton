# SDK演示页面按钮 (btn-activate, btn-deactivate, btn-scan, btn-fly)

**位置**: /demo页面
**类型**: SDK功能演示按钮
**功能**: 测试ShowMeTheButton SDK的核心功能

## 按钮说明

### ✅ 激活光标 (btn-activate)
激活SDK的光标功能。激活后会有一个跟随鼠标的光标替身。

### ⏸️ 停用光标 (btn-deactivate)
停用SDK的光标功能。光标会从屏幕上消失。

### 🔍 扫描页面元素 (btn-scan)
扫描当前页面的所有可交互元素，并在控制台输出扫描结果。

### ✈️ 测试飞向按钮 (btn-fly)
测试SDK的光标飞向目标元素功能。光标会飞向"导出"按钮。

## 状态显示

控制面板显示：
- **SDK状态**: 已加载/未加载
- **光标状态**: 激活/未激活
- **状态消息**: 当前操作提示

## 使用方法

#### 测试光标跟随功能
1. 进入 /demo 页面
2. 等待 SDK 加载完成（显示"SDK已加载"）
3. 点击"激活光标"按钮
4. 移动鼠标，观察小光标跟随
5. 点击"停用光标"关闭

#### 测试飞向功能
1. 先进入 /button-hell 页面（因为飞向目标在那里）
2. 然后回到 /demo 页面
3. 激活光标
4. 点击"测试飞向按钮"
5. 光标会飞到导出按钮并悬停
