# 更多操作菜单 (btn-more-actions)

**位置**: 顶部操作栏 - 新建按钮右侧
**ID**: `btn-more-actions`
**类型**: 下拉菜单按钮
**功能**: 打开包含更多操作的菜单

## 描述

点击"更多操作"按钮会展开一个下拉菜单，包含导入、导出、归档、备份等高级操作。

## 菜单选项

### 📥 导入数据 (btn-import)
从外部文件导入数据到系统。

### 📤 导出数据 (btn-export)
将系统数据导出为文件。点击后会弹出导出对话框，可以选择格式（Excel、PDF、CSV等）。

### 📦 数据归档 (btn-archive)
将不常用的数据移动到归档存储。

### 💾 系统备份 (btn-backup)
创建系统数据的完整备份。

### 🔄 系统恢复 (btn-restore)
从备份中恢复系统数据。

## 使用方法

1. 点击"更多操作"按钮
2. 菜单会下拉显示所有可用操作
3. 点击需要的操作项
4. 某些操作会弹出确认对话框

## 常见用户查询

- "我想导出数据"
- "怎么导入Excel文件？"
- "备份在哪里？"
- "我要导出为PDF"
