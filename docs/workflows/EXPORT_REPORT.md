# 工作流教程：导出报表为PDF

**场景**: 用户需要将当前表格数据导出为PDF文件
**页面**: 按钮地狱 (/button-hell)
**步骤数**: 6步
**预计时间**: 30秒

## 流程图

```
开始 → 点击"更多操作" → 点击"导出数据"
  → 弹出导出对话框 → 选择"PDF"格式
  → 点击"确认导出" → 完成
```

## 详细步骤

### 步骤 1: 进入按钮地狱页面
**操作**: 在导航栏点击"按钮地狱"
**预期结果**: 页面顶部显示操作栏
**相关按钮**: 顶部导航的"🎛️ 按钮地狱"

### 步骤 2: 打开"更多操作"菜单
**操作**: 点击页面顶部的"更多操作 ▾"按钮
**预期结果**: 弹出下拉菜单，显示5个操作选项
**相关按钮**: `btn-more-actions`

### 步骤 3: 选择"导出数据"
**操作**: 在下拉菜单中点击"📤 导出数据"
**预期结果**: 弹出导出对话框
**相关按钮**: `btn-export`

### 步骤 4: 选择PDF格式
**操作**: 在对话框中选择"PDF"单选按钮
**预期结果**: PDF选项被选中（出现选中标记）
**相关元素**: 格式选项 radio button

### 步骤 5: （可选）设置高级选项
**操作**: 点击"⚙️ 高级选项"按钮
**预期结果**: 显示更多配置选项
**相关按钮**: `btn-advanced-options`

### 步骤 6: 确认导出
**操作**: 点击"确认导出"按钮
**预期结果**: 弹出提示"报表已导出为 PDF"，对话框关闭
**相关按钮**: `btn-confirm-export`
