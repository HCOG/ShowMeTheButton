# 工作流教程：创建并发布工作流

**场景**: 用户设计一个简单的审批工作流
**页面**: 工作流设计器 (/workflow)
**步骤数**: 8步
**预计时间**: 5-10分钟

## 流程图

```
开始 → 添加开始节点 → 添加任务节点 → 添加判断节点
  → 添加分支（通过/驳回）→ 添加结束节点
  → 连接节点 → 验证 → 保存 → 发布
```

## 详细步骤

### 步骤 1: 进入工作流设计器
**操作**: 在导航栏点击"工作流"
**预期结果**: 显示工作流设计器界面，默认有6个示例节点
**相关按钮**: 顶部导航的"🔀 工作流"

### 步骤 2: 添加开始节点
**操作**: 在左侧"节点类型"中点击"🟢 开始节点"
**预期结果**: 画布上添加一个绿色开始节点
**相关按钮**: `btn-add-start`

### 步骤 3: 添加任务节点
**操作**: 点击"📝 任务节点"
**预期结果**: 画布上添加一个蓝色任务节点
**相关按钮**: `btn-add-task`

### 步骤 4: 添加判断节点
**操作**: 点击"🔷 判断节点"
**预期结果**: 画布上添加一个黄色判断节点
**相关按钮**: `btn-add-decision`

### 步骤 5: 添加结束节点
**操作**: 点击"🔴 结束节点"
**预期结果**: 画布上添加一个红色结束节点
**相关按钮**: `btn-add-end`

### 步骤 6: 编辑节点属性
**操作**: 点击节点，在右侧属性面板修改
- 节点名称
- 描述
- 处理人

### 步骤 7: 验证工作流
**操作**: 点击"✓ 验证"按钮
**预期结果**: 显示验证结果
- 是否有开始/结束节点
- 节点是否正确连接
- 是否有孤立节点

**相关按钮**: `btn-validate`

### 步骤 8: 保存并发布
**操作**: 
1. 点击"💾 保存"按钮
2. 等待保存成功
3. 点击"🚀 发布"按钮

**预期结果**: 工作流状态变为"已发布"
**相关按钮**: 
- `btn-save`
- `btn-publish`

## 节点连接

- 从一个节点的边缘拖拽到另一个节点即可创建连接
- 判断节点可以有多条输出（通过/驳回）

## 常见用户查询

- "怎么添加节点？"
- "工作流怎么设计？"
- "节点怎么连接？"
- "怎么发布？"
- "验证有什么用？"
- "怎么编辑节点？"

## 辅助功能

- **↩️ 撤销/重做**: 操作历史
- **📋 复制/粘贴**: 快速复用节点
- **🗑️ 删除**: 移除不需要的节点
- **📥 导入/📤 导出**: 备份工作流
