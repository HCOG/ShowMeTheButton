# 保存草稿按钮 (btn-save-draft)

**位置**: 表单顶部
**ID**: `btn-save-draft`
**类型**: 次要操作按钮
**功能**: 保存当前表单为草稿

## 描述

点击此按钮会将当前填写的表单内容保存为草稿，方便稍后继续填写。

## 使用方法

1. 在表单中填写部分内容
2. 点击"💾 保存草稿"按钮
3. 系统会保存当前内容，生成草稿ID
4. 下次可以加载草稿继续填写

## 常见用户查询

- "怎么保存草稿？"
- "我还没填完"
- "下次继续填写"
- "草稿在哪里？"

## 注意事项

- 草稿默认保留30天
- 草稿不能用于正式提交
- 提交表单后草稿会被自动删除
