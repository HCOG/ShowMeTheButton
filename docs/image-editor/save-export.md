# 保存和导出按钮 (btn-save, btn-export, btn-undo, btn-redo)

**位置**: 编辑器顶部
**类型**: 文件操作按钮
**功能**: 保存、导出图片

## 按钮说明

### ↩️ 撤销 (btn-undo)
撤销上一步操作。可以连续撤销多次。

### ↪️ 重做 (btn-redo)
重做被撤销的操作。

### 💾 保存 (btn-save)
保存当前编辑的图片到系统服务器。

### 📤 导出 (btn-export)
将图片导出为本地文件。可以选择格式（PNG、JPG、WEBP）。

## 使用方法

#### 保存图片
1. 完成图片编辑
2. 点击"💾 保存"按钮
3. 系统会保存到服务器
4. 保存成功后会显示提示

#### 导出图片
1. 完成图片编辑
2. 点击"📤 导出"按钮
3. 选择保存位置和格式
4. 浏览器会下载文件
