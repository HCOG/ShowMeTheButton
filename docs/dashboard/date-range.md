# 日期范围选择 (date-range-selector)

**位置**: 仪表盘顶部
**类型**: 时间筛选按钮组
**功能**: 选择数据的时间范围

## 按钮说明

### 今天
显示当天的数据统计。

### 最近7天
显示过去7天的数据统计（默认选项）。

### 最近30天
显示过去30天的数据统计。

### 今年
显示今年以来的数据统计。

## 使用方法

1. 在仪表盘顶部找到日期范围选择器
2. 点击需要的时间范围按钮
3. 选中的按钮会高亮显示
4. 仪表盘数据会自动更新

## 常见用户查询

- "怎么看今天的数据？"
- "我想看最近一个月的"
- "今年的数据"
- "切换时间范围"
