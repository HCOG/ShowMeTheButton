# 生成报表按钮 (btn-generate)

**位置**: 表格工具栏
**ID**: `btn-generate`
**类型**: 次要操作按钮
**功能**: 生成统计报表

## 描述

点击此按钮会根据当前表格数据生成一个统计报表，可以下载或预览。

## 使用方法

1. 在表格中查看要统计的数据
2. 点击"📊 生成报表"按钮
3. 等待系统生成报表
4. 报表生成完成后会提示下载或预览

## 报表类型

系统会根据数据自动生成：
- 数据汇总报表
- 趋势分析图
- 分类统计表

## 常见用户查询

- "怎么生成报表？"
- "我要导出统计"
- "数据汇总在哪里？"
