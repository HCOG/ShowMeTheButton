# Submit Leave Request

## Purpose
Allow an employee to request leave and route the request to their manager.

## Prerequisites
- Employee has remaining leave balance.
- Manager is assigned in the HR profile.

## Procedure
1. Open `/hr/leave` from the HR menu.
2. Choose "New Leave Request".
3. Select the leave type and enter start date and end date.
4. If leave is longer than five business days, add a comment explaining the plan.
5. Submit the request to the manager.
6. Verify the request appears in the leave request table.

## Notes
If the balance is insufficient, contact HR before submitting.
