# 图表类型切换 (chart-type-selector)

**位置**: 销售趋势图标题栏
**类型**: 图表类型选择按钮组
**功能**: 切换不同的图表展示方式

## 按钮说明

### 📈 折线图
显示数据随时间的变化趋势。适合看趋势和走向。

### 📊 柱状图
显示不同时间点的数据对比。适合看量级差异。

### 🥧 饼图
显示各部分占总体的比例。适合看占比。

### 📉 面积图
类似折线图但填充区域。适合看累计量。

## 使用方法

1. 找到"销售趋势"图表
2. 在图表右上角选择图表类型
3. 选中的图表类型会高亮
4. 图表会平滑切换到新类型
