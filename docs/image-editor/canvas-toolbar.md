# 画布操作按钮 (btn-load-image, btn-zoom-in, btn-zoom-out, btn-reset)

**位置**: 画布上方工具栏
**类型**: 画布控制按钮
**功能**: 控制画布的显示和操作

## 按钮说明

### 📂 打开图片 (btn-load-image)
从本地选择一张图片加载到画布。支持JPG、PNG、WEBP格式。

### 🔍+ 放大 (btn-zoom-in)
放大画布视图。每次点击放大25%。

### 🔍- 缩小 (btn-zoom-out)
缩小画布视图。每次点击缩小25%。

### 🔄 重置 (btn-reset)
重置画布到默认大小和位置。

### 适应屏幕 (fitToScreen)
自动调整画布大小以适应屏幕。

## 状态显示

画布工具栏右侧会显示：
- 当前缩放百分比
- 画布大小（宽 × 高）
- 工具状态

## 使用方法

1. 点击"打开图片"加载要编辑的图片
2. 使用放大/缩小按钮调整视图
3. 完成编辑后保存或导出
