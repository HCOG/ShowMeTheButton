# 删除选中按钮 (btn-delete-selected)

**位置**: 表格工具栏 - 左侧
**ID**: `btn-delete-selected`
**类型**: 危险操作按钮
**功能**: 批量删除选中的记录

## 描述

点击此按钮会删除表格中所有已选中的记录。需要先在表格的复选框中选择要删除的项目。

## 使用方法

1. 在表格中找到要删除的记录
2. 勾选记录前面的复选框（可以多选）
3. 点击"🗑️ 删除 (N)"按钮（N 表示选中数量）
4. 系统会弹出确认对话框
5. 确认删除

## 状态说明

- 当没有选中任何记录时，按钮显示为禁用状态
- 选中记录后，按钮显示选中的数量

## 常见用户查询

- "怎么删除选中的数据？"
- "我要批量删除"
- "删除按钮在哪里？"
- "怎么选中多条记录？"

## 注意事项

- ⚠️ 删除操作不可恢复
- 删除前建议先备份数据
- 关联数据也会被同时删除
