# 保存和发布按钮 (btn-save, btn-publish, btn-validate, btn-import, btn-export)

**位置**: 编辑器顶部
**类型**: 工作流管理按钮
**功能**: 保存、发布、导入导出工作流

## 按钮说明

### ↩️ 撤销 (btn-undo)
撤销上一步操作。

### ↪️ 重做 (btn-redo)
重做被撤销的操作。

### 📥 导入 (btn-import)
从文件导入已有的工作流定义（JSON格式）。

### 📤 导出 (btn-export)
将当前工作流导出为文件（JSON格式）。

### ✓ 验证 (btn-validate)
验证工作流是否完整、合法。检查项：
- 是否有开始节点
- 是否有结束节点
- 节点之间是否正确连接
- 是否有孤立的节点

### 💾 保存 (btn-save)
保存当前工作流。状态变为"草稿"。

### 🚀 发布 (btn-publish)
发布工作流。发布后的工作流可以被使用。

## 工作流状态

| 状态 | 含义 |
|------|------|
| 草稿 | 编辑中，未完成 |
| 已保存 | 保存但未发布 |
| 已发布 | 已发布，可被引用 |
| 归档 | 已废弃 |

## 使用方法

#### 发布工作流
1. 完成工作流设计
2. 点击"✓ 验证"检查工作流
3. 修复所有问题
4. 点击"💾 保存"保存为最新版本
5. 点击"🚀 发布"发布工作流
