# 工具栏按钮 (btn-add-start, btn-add-task, btn-add-decision, btn-add-end, btn-add-subprocess)

**位置**: 左侧工具栏
**类型**: 工具选择按钮
**功能**: 选择不同的编辑工具

## 按钮说明

### 🟢 开始节点 (btn-add-start)
添加一个流程的开始节点。开始节点是工作流的入口。

### 📝 任务节点 (btn-add-task)
添加一个任务节点。任务节点用于表示需要执行的操作。

### 🔷 判断节点 (btn-add-decision)
添加一个判断节点。判断节点用于条件分支。

### 🔴 结束节点 (btn-add-end)
添加一个流程的结束节点。结束节点是工作流的出口。

### 📦 子流程 (btn-add-subprocess)
添加一个子流程节点。子流程用于嵌套的复杂流程。

## 工具选择（实时工具）

### ↖️ 选择 (setTool: select)
选择画布上的元素，进行编辑或移动。

### ✂️ 裁剪 (setTool: crop)
裁剪图片到指定区域。

### 🎨 滤镜 (setTool: filter)
应用滤镜效果。

### 📝 文字 (setTool: text)
添加文字图层。

### 🖌️ 画笔 (setTool: brush)
使用画笔工具绘制。

### 🧹 橡皮擦 (setTool: eraser)
擦除部分内容。

### ⬜ 形状 (setTool: shapes)
添加形状（矩形、圆形等）。

### 🔍 缩放 (setTool: zoom)
缩放查看画布。

## 使用方法

1. 在左侧工具栏选择要使用的工具
2. 工具按钮会高亮显示
3. 在画布上使用选中的工具
