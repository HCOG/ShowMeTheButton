# 快速操作按钮 (btn-create-report, btn-schedule-report, btn-share-dashboard, btn-customize)

**位置**: 仪表盘底部 - 快速操作区
**类型**: 操作按钮组
**功能**: 快速执行常用操作

## 按钮说明

### 📊 创建报表 (btn-create-report)
创建一个新的报表。可以选择数据源、时间范围等。

### ⏰ 定时报表 (btn-schedule-report)
设置定时自动生成报表。可以按天、周、月等周期生成。

### 🔗 分享仪表盘 (btn-share-dashboard)
分享当前仪表盘给其他用户。可以设置只读或可编辑权限。

### 🎨 自定义布局 (btn-customize)
自定义仪表盘的布局。可以拖拽组件、调整大小、添加新组件等。

### 📤 导出报表 (btn-export)
导出当前仪表盘为PDF、Excel等格式。

### 🔄 刷新 (btn-refresh)
重新加载仪表盘数据。

## 使用方法

#### 创建定时报表
1. 点击"⏰ 定时报表"按钮
2. 设置报表名称
3. 选择发送频率（每天/每周/每月）
4. 选择接收人（邮件列表）
5. 确认创建

#### 分享仪表盘
1. 点击"🔗 分享仪表盘"按钮
2. 输入接收人邮箱
3. 选择权限（查看/编辑）
4. 添加备注
5. 发送邀请
