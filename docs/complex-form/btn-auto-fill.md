# 自动填充按钮 (btn-auto-fill)

**位置**: 表单顶部
**ID**: `btn-auto-fill`
**类型**: 辅助按钮
**功能**: 自动填充示例数据

## 描述

点击此按钮会自动填充表单的所有字段为示例数据，方便快速查看表单效果或测试。

## 使用方法

1. 在表单顶部找到"🤖 自动填充"按钮
2. 点击按钮
3. 所有表单字段会被填充为示例数据
4. 可以继续修改或直接提交

## 常见用户查询

- "怎么快速填写表单？"
- "示例数据怎么用？"
- "我看到的是一个空表单"

## 注意事项

- 自动填充的数据仅用于演示
- 实际使用时需要填写真实信息
