# 工作流教程：批量删除记录

**场景**: 用户需要批量删除表格中的多条记录
**页面**: 按钮地狱 (/button-hell)
**步骤数**: 5步
**预计时间**: 20秒

## 流程图

```
开始 → 勾选要删除的记录 → 点击"删除"按钮
  → 弹出确认对话框 → 确认删除 → 完成
```

## 详细步骤

### 步骤 1: 选择要删除的记录
**操作**: 在表格中勾选要删除的记录前面的复选框
**说明**: 
- 可以逐个勾选
- 也可以勾选表头的全选框全选所有
- 删除按钮会显示已选中的数量

**相关元素**: 表格复选框 `checkbox-select-all`

### 步骤 2: 点击"删除"按钮
**操作**: 点击表格工具栏的"🗑️ 删除 (N)"按钮（N是选中数量）
**预期结果**: 弹出确认删除对话框
**相关按钮**: `btn-delete-selected`

### 步骤 3: 查看删除确认
**操作**: 在对话框中查看要删除的记录数量
**预期结果**: 显示"确定要删除选中的 N 条记录吗？"
**注意**: 会显示警告"⚠️ 此操作不可恢复！"

### 步骤 4: 确认删除
**操作**: 点击"确认删除"按钮
**预期结果**: 删除执行，对话框关闭
**相关按钮**: `btn-confirm-delete`

### 步骤 5: 查看删除结果
**操作**: 查看表格，确认记录已被删除
**预期结果**: 表格中已无选中的记录，提示"已删除 N 条记录"

## 常见用户查询

- "怎么删除多条记录？"
- "批量删除怎么操作？"
- "删除按钮在哪？"
- "能一次选多条吗？"
- "怎么全选？"
- "删除后能恢复吗？"

## 注意事项

- ⚠️ 删除操作不可恢复
- 建议在删除前先导出备份
- 关联数据也会被删除
