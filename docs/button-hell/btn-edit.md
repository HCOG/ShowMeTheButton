# 表格行操作按钮 (btn-edit-N, btn-copy-N, btn-share-N, btn-download-N, btn-history-N)

**位置**: 表格操作列（每行都有）
**类型**: 行内操作按钮组
**功能**: 对单条记录进行操作

## 描述

表格中每一行都有多个操作按钮，用于对该行记录进行不同的操作。

## 按钮说明

### ✏️ 编辑 (btn-edit-{id})
编辑当前行的记录。点击后会跳转到编辑页面或弹出编辑表单。

### 📋 复制 (btn-copy-{id})
复制当前行记录。系统会创建一个与当前记录相同的新记录，但会重命名以区分。

### 🔗 分享 (btn-share-{id})
分享当前行记录。可以通过链接或邮件分享给其他用户。

### ⬇️ 下载 (btn-download-{id})
下载当前行记录。可以下载为文件或保存到本地。

### 📜 历史 (btn-history-{id})
查看当前行记录的历史修改记录和操作日志。

## 使用方法

1. 在表格中找到要操作的行
2. 在操作列点击相应的按钮
3. 系统会执行相应的操作

## 常见用户查询

- "怎么编辑这条记录？"
- "复制按钮在哪里？"
- "我要分享给同事"
- "怎么下载这个文件？"
- "这条记录修改过几次？"
