# 新建按钮 (btn-add-new)

**位置**: 顶部操作栏 - 最左侧
**ID**: `btn-add-new`
**类型**: 主要操作按钮
**功能**: 创建一个新项目

## 描述

点击"新建"按钮会在系统中创建一个新的项目记录。这是最常用的主要操作按钮之一，通常作为页面的第一个操作选项。

## 使用方法

1. 找到顶部操作栏的"➕ 新建"按钮
2. 点击按钮
3. 系统会弹出创建新项目的表单

## 常见用户查询

- "我想创建一个新项目"
- "怎么新建？"
- "我要新增一个"
- "添加新内容"

## 相关按钮

- 表格内的"添加"按钮（`btn-add-row`）- 在工具栏中也提供添加功能
- "新建用户"按钮（`btn-add-user`）- 在用户管理中
- "添加图层"按钮（`btn-add-layer`）- 在图片编辑器中

## 注意事项

- 只有登录用户才能使用此功能
- 必填字段需要在表单中填写完整
- 创建成功后会自动跳转到新项目的详情页
