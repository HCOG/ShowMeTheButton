# 工作流教程：员工入职信息提交

**场景**: 新员工填写入职登记表
**页面**: 复杂表单 (/complex-form)
**步骤数**: 8步
**预计时间**: 5-10分钟

## 流程图

```
开始 → 填写个人信息 → 下一步
  → 填写地址信息 → 下一步
  → 填写就业信息 → 下一步
  → 上传证件+选择偏好 → 同意条款 → 提交表单
```

## 详细步骤

### 步骤 1: 进入表单页面
**操作**: 在导航栏点击"复杂表单"
**预期结果**: 显示员工入职登记表，默认为第一步
**相关按钮**: 顶部导航的"📝 复杂表单"

### 步骤 2: 填写个人信息（第一步）
**操作**: 填写名、姓、邮箱、手机号等基本信息
**必填项**:
- 名
- 姓
- 邮箱
- 手机号
- 身份证号码

**快捷操作**: 点击"🤖 自动填充"可快速填入示例数据
**相关按钮**: `btn-auto-fill`

### 步骤 3: 进入下一步
**操作**: 点击"下一步 ▶"按钮
**预期结果**: 进入第二步"地址信息"，进度条更新
**相关按钮**: `btn-next`

### 步骤 4: 填写地址信息（第二步）
**操作**: 填写街道、城市、邮编、国家等
**必填项**:
- 街道地址
- 城市
- 邮政编码
- 国家

**可选**: 点击"⚙️ 显示高级选项"填入公寓号、楼层等

### 步骤 5: 填写就业信息（第三步）
**操作**: 填写公司、职位、部门等
**必填项**:
- 公司名称
- 职位
- 部门
- 雇佣类型
- 入职日期

### 步骤 6: 上传证件和选择偏好（第四步）
**操作**:
1. 勾选"我有护照"或"我有驾驶证"（如适用）
2. 点击"📎 上传X扫描件"上传文件
3. 勾选需要的通知方式
4. 选择语言和时区

**相关按钮**: 
- `btn-upload-id`
- `btn-upload-passport`
- `btn-upload-license`

### 步骤 7: 同意服务条款
**操作**: 勾选"我已阅读并同意服务条款和隐私政策"
**说明**: 不勾选无法提交表单

### 步骤 8: 提交表单
**操作**: 点击"✓ 提交表单"按钮
**预期结果**: 显示"表单提交成功！"，表单重置
**相关按钮**: `btn-submit`

- "入职信息在哪？"
- "我要提交员工信息"
- "怎么上传证件？"
- "自动填充怎么用？"
- "草稿能保存吗？"

## 辅助功能

- **保存草稿**: 任何时候点击"💾 保存草稿"可暂存
- **自动填充**: 一键填入示例数据
- **进度条**: 顶部显示当前进度
- **上一步**: 可返回修改之前填的内容
