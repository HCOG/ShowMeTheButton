# 节点类型按钮 (btn-add-start, btn-add-task, btn-add-decision, btn-add-end, btn-add-subprocess)

**位置**: 左侧工具栏 - 节点类型
**类型**: 节点创建按钮组
**功能**: 向工作流画布添加不同类型的节点

## 按钮说明

### 🟢 开始节点 (btn-add-start)
添加流程的开始节点。**每个工作流必须有且只能有一个开始节点**。

### 📝 任务节点 (btn-add-task)
添加一个任务节点。表示需要执行的具体操作。

### 🔷 判断节点 (btn-add-decision)
添加一个判断节点。用于条件分支（如审批是否通过）。

### 🔴 结束节点 (btn-add-end)
添加流程的结束节点。**每个工作流必须有至少一个结束节点**。

### 📦 子流程 (btn-add-subprocess)
添加一个子流程节点。用于嵌套调用其他工作流。

## 节点样式

| 节点类型 | 形状 | 颜色 |
|---------|------|------|
| 开始 | 矩形 | 绿色 |
| 任务 | 矩形 | 蓝色 |
| 判断 | 菱形 | 黄色 |
| 结束 | 矩形 | 红色 |
| 子流程 | 矩形 | 紫色 |

## 使用方法

1. 在左侧工具栏"节点类型"部分
2. 点击需要添加的节点类型
3. 节点会出现在画布上
4. 可以拖拽节点调整位置
5. 点击节点可以编辑属性
