| 教程 | 场景 | 步骤数 | 预计时间 |

## 教程格式说明

每个教程包含以下信息：

1. **场景**: 描述使用场景
2. **流程图**: 简化版的步骤流程
3. **详细步骤**: 每一步的具体操作、预期结果和相关按钮ID
4. **常见用户查询**: 用户可能问的问题
5. **辅助功能**: 相关联的辅助按钮
