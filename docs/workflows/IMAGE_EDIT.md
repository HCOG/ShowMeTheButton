# 工作流教程：编辑并保存图片

**场景**: 用户打开图片，应用滤镜，保存
**页面**: 图片编辑器 (/image-editor)
**步骤数**: 7步
**预计时间**: 1-2分钟

## 流程图

```
开始 → 打开图片 → 选择滤镜 → 调整参数
  → 保存或导出 → 完成
```

## 详细步骤

### 步骤 1: 进入图片编辑器
**操作**: 在导航栏点击"图片编辑"
**预期结果**: 显示图片编辑器界面
**相关按钮**: 顶部导航的"🎨 图片编辑"

### 步骤 2: 打开图片
**操作**: 点击画布工具栏的"📂 打开图片"按钮
**预期结果**: 弹出文件选择对话框
**相关按钮**: `btn-load-image`

### 步骤 3: 选择文件
**操作**: 选择本地图片文件（JPG、PNG、WEBP）
**预期结果**: 图片加载到画布上

### 步骤 4: 选择滤镜
**操作**: 在左侧工具栏"滤镜"区域选择滤镜
**选项**:
- 📜 复古 (`btn-filter-vintage`)
- ⬛ 黑白 (`btn-filter-bw`)
- 💫 模糊 (`btn-filter-blur`)
- ✨ 锐化 (`btn-filter-sharpen`)

**相关按钮**: `btn-filter-*`

### 步骤 5: 调整参数（可选）
**操作**: 在"调整"区域调整参数
- ☀️ 亮度 (`btn-brightness`)
- ◐ 对比度 (`btn-contrast`)
- 🎨 饱和度
- 🌈 色相

### 步骤 6: 保存到服务器
**操作**: 点击顶部"💾 保存"按钮
**预期结果**: 图片保存到服务器
**相关按钮**: `btn-save`

### 步骤 7: 导出到本地（可选）
**操作**: 点击顶部"📤 导出"按钮
**预期结果**: 浏览器下载图片到本地
**相关按钮**: `btn-export`

## 辅助功能

- **↩️ 撤销**: 撤销上一步操作
- **↪️ 重做**: 重做被撤销的操作
- **🔍 放大/缩小**: 调整视图
- **🔄 重置**: 恢复初始状态
