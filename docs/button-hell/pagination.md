# 分页按钮 (btn-prev-page, btn-next-page, btn-first-page, btn-last-page)

**位置**: 表格底部
**类型**: 导航按钮组
**功能**: 在表格页面之间导航

## 描述

当表格数据超过一页时，会显示分页按钮组，可以跳转到不同页面。

## 按钮说明

### ◀ 上一页 (btn-prev-page)
返回上一页。第一页时禁用。

### 下一页 ▶ (btn-next-page)
进入下一页。最后一页时禁用。

### ⏮ 首页 (btn-first-page)
跳转到第一页。

### ⏭ 末页 (btn-last-page)
跳转到最后一页。

## 状态显示

- 当前页码/总页数
- 选中数量
- 选中状态指示

## 常见用户查询

- "怎么翻页？"
- "下一页在哪？"
- "我要回到第一页"
- "跳到最后一页"
