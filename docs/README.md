# Knowledge Base Documentation

This directory contains the knowledge base documentation for the RAG system.

## Structure

```
docs/
├── button-hell/           # Button Hell page docs
│   ├── export-button.md
│   ├── save-button.md
│   ├── delete-button.md
│   └── ...
│
├── complex-form/          # Complex Form page docs
│   ├── submit-button.md

│   └── ...
│
├── image-editor/          # Image Editor page docs
│   ├── crop-tool.md
│   ├── filter-tool.md
│   └── ...
│
└── dashboard/            # Dashboard page docs
    └── ...
```

## Purpose

These markdown files serve as the source documents for the RAG knowledge base.
Each file documents a specific UI component and its functionality.

## Format

Each markdown file should include:

- Component name and ID
- Functionality description
- Usage instructions
- Related components
- Example user queries

See main [README.md](../README.md) for more information.
